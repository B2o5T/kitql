# Welcome to the doc!

You want to discover, watch it: [Youtube - KitQL - SvelteKit/GraphQL](https://www.youtube.com/watch?v=6pH4fnFN70w)

## Try the demo 1

```bash
# Get the first demo app (will create a folder: kitql-demo1)
npx degit github:jycouet/kitql/examples/demo1 kitql-demo1

# Navigate to the new folder
cd kitql-demo1

# Install everything
yarn

# Run the app ⚡⚡⚡
yarn dev
```
